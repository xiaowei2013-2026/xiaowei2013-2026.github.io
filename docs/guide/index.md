## 如何使用

### 浏览内容

- 使用左侧**侧边栏**快速导航到不同分类
- 使用顶部**搜索栏**（快捷键 `Ctrl+K`）搜索任意内容
- 查看文章右侧的**页面导航**快速跳转到具体章节

### 阅读体验

- 点击右上角 🌙 图标切换**暗色模式**
- 每篇文章底部有**上一页/下一页**导航
- 支持**移动端**响应式布局
