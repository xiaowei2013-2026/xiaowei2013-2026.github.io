# 第一步总是最重的

<Badge type="tip" text="2026-06-20" />

想建一个个人知识库这件事，想了很久。

每次看到别人漂亮的博客、井井有条的笔记系统，心里就想：我也该搞一个。然后打开终端，然后关掉，然后打开，然后关掉。反复了好几个来回。

不是不会，是不敢。怕搞出来没人看，又怕有人看；怕写的东西太浅被人笑，又怕浪费时间去学这些"花里胡哨的工具"。

但今天终于迈了一步。

其实也没那么难。npm install，改几行配置，写一篇 Markdown — 20 分钟的事。难的不是技术，是心里那个"万一不值得呢"的声音。

### 💡 一个小想法

> 任何一件事，只要开始做的那个瞬间，就不算浪费。

第一步总是最重的。但迈出去之后，第二步就轻了。
