# 2026-06-29
昨天（6.28）收到了硕士双证，没有想象中的激动。收到的时候非常平静，看了几眼后就收起了。现在回头看，走得好匆忙。6月2号（周二），租了学位服，与李、袁、导师、组里同学合影，在校园里面合影， 晚上和导师一起吃饭。6月4号（周四）晚上，与钊煌、志哥一起吃了晚饭。6月5号（周五）晚上离校，离别前与两位舍友合照。就这样已经毕业。6月9号（周二），以实习生的身份入职滴滴。中间没有参加学院与学校的毕业典礼，也没有拍很多照留念，也没有一直呆在学校，与导师的联系更是很少，自从去年6月开完最后一次组会到现在，中间与导师很少有沟通，也很少见面，当然不止我这样，李袁他们大概率也是这样。硕士的毕业好像特别匆忙，有的人走得特别早，有的人非必要不返校。或许大家都是成人了，有更多重要的事需要做。我现在还没有转变过身份，整天还是很懒散，像没有成长起来的学生。现在感觉就是当时太过匆忙，没有好好的在吉大转转，像是有什么事没做完，对于山师就没有这种感觉。后面可能永远都不会再回吉大了，除非特意回，山师则不同。原因也很简单，山师的地理位置可以，离家近、离工作的地方也近，一个人去山师校园看看也不会觉得孤单，原因是山师校园里有生活过的痕迹。吉大则不同，地理位置不好，在一个角落里。吉大校园里也没有生活过的痕迹，三年的硕士生活，二年是995，在实验室度过，即使周末有空也是在寝室。第三年则是一直在忙着找工作，直到最后才找到一个还说得上去的工作（不算好，绝对不算好），所以吉大真没有生活过的痕迹，很多地方、很多东西都没有接触过。与本科相比，硕士显得太匆忙，没有来得及好好回味就结束了。毕业照也是，自愿参加，这与本科形成强烈对比。向前看吧，别一直挂着不放，没什么意义。要让自己忙碌起来，周末多出去转转，把小区先熟悉熟悉，再把附近熟悉熟悉，多与人沟通，这样才是正常的人。
